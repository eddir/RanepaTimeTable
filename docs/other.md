# Другие календари

Выберите группу:

- [УЦП-23](#уцп-23)

## УЦП-23

1. Откройте календарь.
2. Найдите пункт "Добавить календарь по ссылке".
3. Введите адрес календаря:

```plaintext-vue
{{ calendarUrl }}/timetable.ics
```

4. Нажмите "Добавить".
